Detailed Description

The RcvMeetingChatController class is a managing class to control the in-meeting chat related functions and states in an active meeting.

A successful call of sendMessageToAll triggers the onChatMessageSend event callback for the local user, the remote users will retrieve the onChatMessageReceived event notification.
